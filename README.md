## インストール方法
1. .NET デスクトップ ランタイムをインストールしていない場合は、コマンドプロンプトで管理者として以下を実行します:
```
winget install Microsoft.DotNet.Runtime.9
```
3. turnedit-setup.exe をダウンロードします。
4. turnedit-setup.exe を管理者として実行します。
